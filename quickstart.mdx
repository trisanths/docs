<Steps>
  <Step title="Sign up for an account">
    Visit our <Link href="https://app.cyrionlabs.com">application portal</Link> and register your nonprofit organization. You'll need to provide:
    
    <ul>
      <li>Organization name and details</li>
      <li>Proof of nonprofit status (501(c)(3) or equivalent)</li>
      <li>Intended use case for AI</li>
    </ul>
    
    Once approved, you'll receive your API key instantly.
  </Step>

  <Step title="Install the SDK">
    Choose your preferred programming language and install our SDK:
    
    <Tabs>
      <Tab title="Python">
        ```bash
        pip install cyrionai
        ```
      </Tab>
      <Tab title="Node.js">
        ```bash
        npm install cyrionai
        ```
      </Tab>
      <Tab title="cURL">
        No installation needed - use cURL directly
      </Tab>
    </Tabs>
  </Step>

  <Step title="Make your first API call">
    Test the API with a simple chat completion:
    
    <Tabs>
      <Tab title="Python">
        ```python
        import cyrionai
        
        client = cyrionai.CyrionAI(api_key="your-api-key")
        
        response = client.chat.completions.create(
            model="gpt-4o-mini",
            messages=[
                {"role": "user", "content": "Hello! How can AI help nonprofits?"}
            ]
        )
        
        print(response.choices[0].message.content)
        ```
      </Tab>
      <Tab title="Node.js">
        ```javascript
        const CyrionAI = require('cyrionai');
        
        const client = new CyrionAI({
            apiKey: 'your-api-key'
        });
        
        const response = await client.chat.completions.create({
            model: 'gpt-4o-mini',
            messages: [
                {role: 'user', content: 'Hello! How can AI help nonprofits?'}
            ]
        });
        
        console.log(response.choices[0].message.content);
        ```
      </Tab>
      <Tab title="cURL">
        ```bash
        curl https://api.cyrionlabs.com/v1/chat/completions \
          -H "Authorization: Bearer your-api-key" \
          -H "Content-Type: application/json" \
          -d '{
            "model": "gpt-4o-mini",
            "messages": [
              {"role": "user", "content": "Hello! How can AI help nonprofits?"}
            ]
          }'
        ```
      </Tab>
    </Tabs>
  </Step>
</Steps>

## Authentication

All API requests require authentication using your API key. Include it in the Authorization header:

```bash
Authorization: Bearer your-api-key
```

<Note>
  Keep your API key secure and never share it publicly. If you suspect your key has been compromised, regenerate it immediately in your dashboard.
</Note>

## Base URL

All API requests should be made to:

```
https://api.cyrionlabs.com
```

## Rate Limits

CyrionAI provides generous rate limits for nonprofit organizations:

- **Chat Completions:** 100 requests per minute
- **Image Generation:** 20 requests per minute  
- **Video Generation:** 10 requests per minute
- **Audio Transcription:** 50 requests per minute

<Note>
  Rate limits are per API key. Contact us if you need higher limits for your use case.
</Note>

## Error Handling

The API returns standard HTTP status codes. Common errors include:

- `401 Unauthorized` - Invalid or missing API key
- `429 Too Many Requests` - Rate limit exceeded
- `400 Bad Request` - Invalid request parameters
- `500 Internal Server Error` - Server error

Example error response:

```json
{
  "error": {
    "message": "Invalid API key",
    "type": "invalid_request_error",
    "code": "invalid_api_key"
  }
}
```
